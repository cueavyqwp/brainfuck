# 开始

首先在`VSCode`中安装扩展 [Brainfuck Syntax](https://github.com/attilabuti/brainfuck-syntax)

# 语法

|||
:---:|:---:
`>`|指针向右移
`<`|指针向左移
`+`|在指针处加
`-`|在指针处减
`.`|输出当前指针指向内容
`,`|输入当内容向当前指针
`[`|循环开始
`]`|循环末尾

# 运行

打开编写好的`brainfuck`文件按`F5`运行
